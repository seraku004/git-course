おさらいコマンド一覧ページ

Add

指定したファイルをローカルのリポジトリに登録する準備をするためのコマンド

使い方

特定のファイルだけ登録する
git add ファイル名

指定したファイルを登録する。
git add ファイル１ ファイル２...

フォルダ内のすべてを登録する。
git add -A git add --all git add .

特定の拡張子のものだけ登録する。
git add *.拡張子

Commit

Addされたファイルをローカルリポジトリに登録します。
※履歴として残るコメントが必須
※実行前にuser.nameとuser.emailの設定が必須

使い方

git commit -m "コミットメッセージ"

Status

現時点のローカルディレクトリの状態を表示します。

使い方

git status
